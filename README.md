# pickle-pe
Usage : pickle-pe.exe <file.exe> -parameter
![Screenshotpng](https://user-images.githubusercontent.com/10811344/119184504-1ec7fa80-ba7e-11eb-9f43-4d58853c5f48.png)


-fi --File Information  
-sh --Section Header  
-ih --Image File Header  
-dh --Dos Header  
-nt --NT Header  
-oh --Optional Header  
-dd --Data Directory  
-id --Import Directory  
-h --Hex Dump  
-s --Strings  

--help  
--version
